# Aspen Plus Distillation Optimization Using Python

## Project Overview

This project investigates the optimization of a binary distillation column using Aspen Plus V14 and Python.

A benzene–toluene mixture is separated using a RadFrac distillation column. Aspen Plus is used to perform a sensitivity analysis on reflux ratio, while Python is used to analyze the exported simulation data and determine how product purity affects energy consumption.

The project demonstrates process simulation, engineering optimization, and data analysis.

---

## Objectives

- Build a RadFrac distillation model in Aspen Plus
- Perform a sensitivity analysis on reflux ratio
- Export simulation data to CSV
- Analyze results using Python
- Visualize energy consumption versus product purity
- Identify the point of diminishing returns ("knee point")
